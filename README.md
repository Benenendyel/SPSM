# Supermarket Product Management System

The **Supermarket Product Management System** is a locally-hosted, browser-based solution designed to streamline in-store operations. It provides a seamless experience for customers, cashiers, and store administrators through three dedicated interfaces:

---

# Supports three interfaces

1. Customer Scanning Stations - can only view product details
2. Cashier Terminal - can handle checkouts
3. Administrator Dashboard - can do full CRUD

---

# Tools we used

1. NodeJS
2. JS (HTML, CSS)
3. SQLLite - for portability of
