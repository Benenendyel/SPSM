# Supermarket Product Management System

A locally-hosted, browser-based solution designed to streamline in-store operations with dedicated interfaces for customers, cashiers, and administrators.

## Features

The system provides three specialized interfaces to support different user roles:

### Customer Scanning Stations

- View product details and information
- Self-service product lookup
- Read-only access for customer convenience

### Cashier Terminal

- Process customer checkouts efficiently
- Access product database for pricing
- Handle transaction management

### Administrator Dashboard

- Full CRUD operations (Create, Read, Update, Delete)
- Manage product inventory
- Configure system settings
- Monitor store operations

---

## Tech Stack

- **Backend:** Node.js
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite (for portability)

## Getting Started

### Prerequisites

- Node.js installed on your system
- Modern web browser (Chrome, Firefox, Edge, or Safari)

### Installation

Clone the repository:

```bash
git clone https://github.com/Benenendyel/SPSM
```

Navigate to project directory:

```bash
cd SPSM
```

Install dependencies:

```bash
npm install
```

Start the application:

```bash
npm start
```

### Usage

Once the application is running, visit this your localhost with the port 3000:

- Homepage: `http://localhost:3000`
